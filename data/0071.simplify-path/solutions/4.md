## [71.简化路径 中文热门题解4](https://leetcode.cn/problems/simplify-path/solutions/100000/python-4-line-by-qqqun902025048)

作者：[QQqun902025048](https://leetcode.cn/u/QQqun902025048)
```
class Solution:
    def simplifyPath(self, path: str) -> str:
        r = []
        for s in path.split('/'):
            r = {'':r, '.':r, '..':r[:-1]}.get(s, r + [s])
        return '/' + '/'.join(r)
```
