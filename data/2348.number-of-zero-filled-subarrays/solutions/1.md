考虑每个以 $0$ 结尾的子数组的个数。

做法：统计连续 $0$ 组成的长度 $c$，每个 $c$ 可以贡献 $c$ 个子数组。

```py [sol1-Python3]
class Solution:
    def zeroFilledSubarray(self, nums: List[int]) -> int:
        ans = c = 0
        for x in nums:
            if x:
                c = 0
            else:
                c += 1
                ans += c
        return ans
```

```java [sol1-Java]
class Solution {
    public long zeroFilledSubarray(int[] nums) {
        var ans = 0L;
        var c = 0;
        for (var num : nums)
            if (num != 0) c = 0;
            else ans += ++c;
        return ans;
    }
}
```

```cpp [sol1-C++]
class Solution {
public:
    long long zeroFilledSubarray(vector<int> &nums) {
        long ans = 0L;
        int c = 0;
        for (int num : nums)
            if (num) c = 0;
            else ans += ++c;
        return ans;
    }
};
```
