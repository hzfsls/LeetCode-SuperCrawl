# 思路

# Code

时间180 ms击败94.35%；内存49.9 MB击败52.93%

```Python3 []
class Solution:
    def firstCompleteIndex(self, arr: List[int], mat: List[List[int]]) -> int:
        m, n, dct = len(mat), len(mat[0]), defaultdict(tuple)
        for i in range(m):
            for j in range(n):
                dct[mat[i][j]] = (i, j)
        row, col = [0] * m, [0] * n
        for i, a in enumerate(arr):
            r, c = dct[a]
            row[r] += 1
            col[c] += 1
            if row[r] == n or col[c] == m:
                return i
```
