# using的妙用
此题不难，通过`(7, 2018)的净现值不在 NPV 表中, 我们把它看作是 0`可以知道要使用left join，
未匹配的npv为null，用ifnull判断即可。
巧妙的是使用using（）可以直接写多个字段！

### 代码

```mysql
# Write your MySQL query statement below
select
    q.id,
    q.year,
    ifnull(npv,0) npv
from
    Queries q
    left join
    NPV n using(id,year);

```
