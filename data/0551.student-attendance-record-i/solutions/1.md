## [551.学生出勤记录 I 中文热门题解1](https://leetcode.cn/problems/student-attendance-record-i/solutions/100000/gong-shui-san-xie-jian-dan-mo-ni-ti-by-a-hui7)

作者：[AC_OIer](https://leetcode.cn/u/AC_OIer)
## 模拟

放大假，根据题意进行模拟即可。

代码：
```Java []
class Solution {
    public boolean checkRecord(String s) {
        int n = s.length();
        char[] cs = s.toCharArray();
        for (int i = 0, cnt = 0; i < n; ) {
            char c = cs[i];
            if (c == 'A') {
                cnt++;
                if (cnt >= 2) return false;
            } else if (c == 'L') {
                int j = i;
                while (j < n && cs[j] == 'L') j++;
                int len = j - i;
                if (len >= 3) return false;
                i = j;
                continue;
            }
            i++;
        }
        return true;
    }
}
```
* 时间复杂度：$O(n)$
* 空间复杂度：使用 `charAt` 代替 `toCharArray` 的话为 $O(1)$，否则为 $O(n)$
