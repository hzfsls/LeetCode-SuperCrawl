# 思路

一次循环，求前后缀集合，返回差值。

# Code

时间68 ms击败100%；内存16.1 MB击败100%

```Python3 []
class Solution:
    def distinctDifferenceArray(self, nums: List[int]) -> List[int]:
        return [len(set(nums[:i])) - len(set(nums[i:])) for i in range(1, len(nums) + 1)]
```
