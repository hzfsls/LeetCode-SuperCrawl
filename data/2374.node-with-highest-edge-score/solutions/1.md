```py [sol1-Python3]
class Solution:
    def edgeScore(self, edges: List[int]) -> int:
        ans, score = 0, [0] * len(edges)
        for i, to in enumerate(edges):
            score[to] += i
            if score[to] > score[ans] or score[to] == score[ans] and to < ans:
                ans = to
        return ans
```
