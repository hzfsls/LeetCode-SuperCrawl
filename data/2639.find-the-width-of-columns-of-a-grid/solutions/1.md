# 思路

**第一步：转置二维矩阵grid**

**第二步：遍历转置后的各行，比较长度，取最大值**

**第三步：返回各行最大值的结果列表**


# Code

python一行即可，双百：

时间48 ms击败100%；内存16.4 MB击败100%

```Python3 []
class Solution:
    def findColumnWidth(self, grid: List[List[int]]) -> List[int]:
        return [max(len(str(x)) for x in lst) for lst in zip(*grid)]
```
