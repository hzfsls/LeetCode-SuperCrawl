### 代码

```mysql
# Write your MySQL query statement below
SELECT
    transaction_id
FROM transactions
WHERE (DATE(day), amount) in (
SELECT 
    DATE(day), MAX(amount)
FROM 
    transactions 
GROUP BY DATE(day)
HAVING amount = MAX(amount)
)
ORDER BY transaction_id;
```
