## 方法：模拟


**思路及算法**
考虑模拟：
1.用数组 cnts 保存两个字符串的字母差值
2.统计差值之和，用绝对值即可，不用考虑谁多谁少的问题


**代码**

```java
class Solution {
  public int minSteps(String s, String t) {
    int[] cnts = new int[26];
    for (char c : s.toCharArray()) {
      cnts[c - 'a']++;
    }
    for (char c : t.toCharArray()) {
      cnts[c - 'a']--;
    }
    int res = 0;
    for (int i : cnts) res += Math.abs(i);
    return res;
  }
}
```


**复杂度分析**
- 时间复杂度：O(n), n 为max(s.length, t.length)
- 空间复杂度：O(C), C 为26
