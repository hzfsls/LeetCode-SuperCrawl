## [1572.矩阵对角线元素的和 中文热门题解1](https://leetcode.cn/problems/matrix-diagonal-sum/solutions/100000/yong-shi-chao-guo-100-by-fcatkzzo6n)

### 解题思路
此处撰写解题思路
加完主对角线后，将其置为零
### 代码

```python3
class Solution:
    def diagonalSum(self, mat: List[List[int]]) -> int:
        n = len(mat)
        ans = 0
        for i in range(n):
            ans += mat[i][i]
            mat[i][i] = 0
            ans += mat[i][n-i-1]
        
        return ans
```
